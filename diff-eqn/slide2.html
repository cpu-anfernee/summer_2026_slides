Chapter 1.2

Solutions and Initial Value Problems

n-th Order Ordinary Differential Equation

An nth-order ODE is an equality relating the independent variable to the nth derivative (and usually lower-order derivatives as well) of the dependent variable.

General form:

\[ F\left(x, y, \frac{dy}{dx}, \dots, \frac{d^{n}y}{dx^{n}}\right) = 0 \]

where \(F\) is a function that depends on \(x\), \(y\), and the derivatives of \(y\) up to order \(n\).

Explicit Solution

Definition 1: A function \( \phi(x) \) that when substituted for \( y \) in the differential equation satisfies the equation for all \( x \) is called an explicit solution to the equation.

Example: \( \phi(x) = x^{2} - x^{-1} \) is an explicit solution to the linear equation:

\[ \frac{d^{2}y}{dx^{2}} - \frac{2}{x^{2}}y = 0 \]

Implicit Solution

Definition 2: A relation \( G(x, y) = 0 \) is said to be an implicit solution to the differential equation if it defines one or more explicit solutions.

Example: \( 4x^{2} - y^{2} = C \) is an implicit solution to:

\[ y\frac{dy}{dx} - 4x = 0 \]

Initial Value Problem (IVP)

Definition 3: Find a solution to the differential equation that satisfies at \( x_{0} \) the \( n \) initial conditions:

\[ y(x_{0}) = y_{0}, \] \[ \frac{dy}{dx}(x_{0}) = y_{1}, \] \[ \dots \] \[ \frac{d^{n-1}y}{dx^{n-1}}(x_{0}) = y_{n-1} \]

where \( x_{0} \) and \( y_{0}, y_{1}, \dots, y_{n-1} \) are given constants.

Initial Value Problem Example

Show that \( \phi(x) = \sin x - \cos x \) is a solution to the initial value problem:

\[ \frac{d^{2}y}{dx^{2}} + y = 0 \]

Subject to the initial conditions:

\[ y(0) = -1, \quad \frac{dy}{dx}(0) = 1 \]
