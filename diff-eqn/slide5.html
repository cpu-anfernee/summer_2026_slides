Chapter 2.3

Linear First-Order Equations

Solving using the Integrating Factor Method

Standard Form

A first-order differential equation is linear if it can be expressed in the form:

\[ a_1(x)\frac{dy}{dx} + a_0(x)y = b(x) \]

The Standard Form

To solve the equation, we must divide by the leading coefficient \( a_1(x) \) to get it into the standard form:

\[ \frac{dy}{dx} + P(x)y = Q(x) \]

The Integrating Factor Method

  1. Standard Form: Write the equation as \( \frac{dy}{dx} + P(x)y = Q(x) \).
  2. Find the Integrating Factor: Calculate \( \mu(x) \) (or \( I(x) \)):
    \[ \mu(x) = e^{\int P(x) dx} \]
  3. Multiply: Multiply the entire standard form equation by \( \mu(x) \). The left side will always collapse into the derivative of a product:
    \[ \frac{d}{dx}[\mu(x) y] = \mu(x) Q(x) \]
  4. Integrate: Integrate both sides with respect to \( x \):
    \[ \mu(x) y = \int \mu(x) Q(x) dx + C \]
  5. Solve for y: Divide by \( \mu(x) \) to find the general explicit solution.

Example 1: Basic Standard Form

Solve: \( \frac{dy}{dx} + 2xy = x \)

  1. Identify \( P(x) \): The equation is already in standard form. \( P(x) = 2x \).
  2. Find the Integrating Factor:
    \( \mu(x) = e^{\int 2x \, dx} = e^{x^2} \)
    (We drop the constant of integration here).
  3. Multiply and Collapse:
    \( e^{x^2}\frac{dy}{dx} + 2xe^{x^2}y = xe^{x^2} \implies \frac{d}{dx}[e^{x^2} y] = xe^{x^2} \)
  4. Integrate Both Sides: Use \( u \)-substitution (\( u=x^2 \)) on the right.
    \( e^{x^2} y = \int xe^{x^2} dx = \frac{1}{2}e^{x^2} + C \)
  5. Final Answer: Divide by \( e^{x^2} \).
    \( y = \frac{1}{2} + C e^{-x^2} \)

Example 2: Division Required

Solve: \( x \frac{dy}{dx} + 2y = x^3 \), for \( x > 0 \)

  1. Standard Form: Divide everything by \( x \).
    \( \frac{dy}{dx} + \frac{2}{x} y = x^2 \). So, \( P(x) = \frac{2}{x} \).
  2. Find the Integrating Factor:
    \( \mu(x) = e^{\int \frac{2}{x} dx} = e^{2\ln x} = e^{\ln x^2} = x^2 \)
  3. Multiply and Collapse:
    \( x^2 \frac{dy}{dx} + 2xy = x^4 \implies \frac{d}{dx}[x^2 y] = x^4 \)
  4. Integrate Both Sides:
    \( x^2 y = \int x^4 dx = \frac{x^5}{5} + C \)
  5. Final Answer: Divide by \( x^2 \).
    \( y = \frac{x^3}{5} + \frac{C}{x^2} \)

Example 3: Constant Coefficients

Solve: \( \frac{dy}{dx} - 3y = 6 \)

  1. Identify \( P(x) \): \( P(x) = -3 \).
  2. Find the Integrating Factor:
    \( \mu(x) = e^{\int -3 dx} = e^{-3x} \)
  3. Multiply and Collapse:
    \( e^{-3x}\frac{dy}{dx} - 3e^{-3x}y = 6e^{-3x} \implies \frac{d}{dx}[e^{-3x} y] = 6e^{-3x} \)
  4. Integrate Both Sides:
    \( e^{-3x} y = \int 6e^{-3x} dx = -2e^{-3x} + C \)
  5. Final Answer: Multiply by \( e^{3x} \).
    \( y = -2 + C e^{3x} \)

Example 4: Trigonometric Functions

Solve: \( \frac{dy}{dx} + (\cot x) y = \csc x \)

  1. Identify \( P(x) \): \( P(x) = \cot x \).
  2. Find the Integrating Factor:
    \( \mu(x) = e^{\int \cot x \, dx} = e^{\ln|\sin x|} = \sin x \)
  3. Multiply and Collapse:
    \( \sin x \frac{dy}{dx} + \cos x \cdot y = \csc x \sin x \)
    \( \frac{d}{dx}[y \sin x] = 1 \)
    (Note: \(\csc x \cdot \sin x = 1\))
  4. Integrate Both Sides:
    \( y \sin x = \int 1 dx = x + C \)
  5. Final Answer: Divide by \( \sin x \).
    \( y = x \csc x + C \csc x \)

Example 5: Integration by Parts Needed

Solve: \( x \frac{dy}{dx} + y = x e^x \)

  1. Standard Form: Divide by \( x \).
    \( \frac{dy}{dx} + \frac{1}{x} y = e^x \). So, \( P(x) = \frac{1}{x} \).
  2. Find the Integrating Factor:
    \( \mu(x) = e^{\int \frac{1}{x} dx} = e^{\ln x} = x \)
  3. Multiply and Collapse: We actually just multiplied it back to the original form!
    \( x \frac{dy}{dx} + y = x e^x \implies \frac{d}{dx}[xy] = x e^x \)
  4. Integrate Both Sides: Use Integration by Parts for \( \int x e^x dx \).
    \( u = x, \ dv = e^x dx \implies du = dx, \ v = e^x \)
    \( xy = x e^x - \int e^x dx \implies xy = x e^x - e^x + C \)
  5. Final Answer: Divide by \( x \).
    \( y = e^x - \frac{e^x}{x} + \frac{C}{x} \)

Example 6: Initial Value Problem

Solve: \( \frac{dy}{dx} + y = e^{3x} \), given \( y(0) = 4 \)

  1. Identify \( P(x) \) and Integrating Factor:
    \( P(x) = 1 \implies \mu(x) = e^{\int 1 dx} = e^x \)
  2. Multiply and Collapse:
    \( e^x \frac{dy}{dx} + e^x y = e^{4x} \implies \frac{d}{dx}[ye^x] = e^{4x} \)
  3. Integrate and Find General Solution:
    \( y e^x = \int e^{4x} dx = \frac{1}{4}e^{4x} + C \)
    \( y = \frac{1}{4}e^{3x} + C e^{-x} \)
  4. Apply Initial Condition: Plug in \( x=0, y=4 \).
    \( 4 = \frac{1}{4}e^0 + C e^0 \implies 4 = \frac{1}{4} + C \implies C = \frac{15}{4} \)
  5. Particular Solution:
    \( y = \frac{1}{4}e^{3x} + \frac{15}{4}e^{-x} \)
