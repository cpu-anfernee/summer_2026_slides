Chapter 7.4

Inverse Laplace Transform

Returning to the Time Domain

The Inverse Operator

If $F(s)$ represents the Laplace transform of a function $f(t)$, then $f(t)$ is the inverse Laplace transform of $F(s)$.

\[ \mathcal{L}^{-1}\{F(s)\} = f(t) \]

Linearity Property

Just like the forward transform, the inverse Laplace transform is a linear operator. Constants can be pulled out, and sums/differences can be split apart:

\[ \mathcal{L}^{-1}\{\alpha F(s) + \beta G(s)\} = \alpha \mathcal{L}^{-1}\{F(s)\} + \beta \mathcal{L}^{-1}\{G(s)\} \]

Table of Inverse Transforms

We use the exact same table from Chapter 7.2, but we read it from right to left (from the $s$-domain back to the $t$-domain).

$\mathcal{L}^{-1}\left\{\frac{1}{s}\right\} = 1$

$\mathcal{L}^{-1}\left\{\frac{n!}{s^{n+1}}\right\} = t^n$

$\mathcal{L}^{-1}\left\{\frac{1}{s-a}\right\} = e^{at}$

$\mathcal{L}^{-1}\left\{\frac{b}{s^2 + b^2}\right\} = \sin bt$

$\mathcal{L}^{-1}\left\{\frac{s}{s^2 + b^2}\right\} = \cos bt$

$\mathcal{L}^{-1}\{F(s-a)\} = e^{at}f(t)$

The hardest part of 7.4 is algebraically manipulating $F(s)$ so it perfectly matches these forms!

Example 1: Splitting and Adjusting

Evaluate: $\mathcal{L}^{-1}\left\{ \frac{3s+5}{s^2+7} \right\}$

  1. Split the Fraction:
    $\mathcal{L}^{-1}\left\{ \frac{3s}{s^2+7} + \frac{5}{s^2+7} \right\}$
  2. Apply Linearity: Pull out constants.
    $= 3\mathcal{L}^{-1}\left\{ \frac{s}{s^2+7} \right\} + 5\mathcal{L}^{-1}\left\{ \frac{1}{s^2+7} \right\}$
  3. Identify Forms ($b^2 = 7 \implies b = \sqrt{7}$):
    The first term matches cosine perfectly. The second term is missing $b$ in the numerator to be sine.
  4. Multiply and Divide by $b$:
    $= 3\mathcal{L}^{-1}\left\{ \frac{s}{s^2+(\sqrt{7})^2} \right\} + \frac{5}{\sqrt{7}}\mathcal{L}^{-1}\left\{ \frac{\sqrt{7}}{s^2+(\sqrt{7})^2} \right\}$
  5. Final Answer:
    $= 3\cos(\sqrt{7}t) + \frac{5}{\sqrt{7}}\sin(\sqrt{7}t)$

Example 2: Partial Fractions

Evaluate: $\mathcal{L}^{-1}\left\{ \frac{s+1}{s^2 - 4s - 5} \right\}$

  1. Factor the Denominator:
    $s^2 - 4s - 5 = (s-5)(s+1)$
  2. Wait, simplify first!
    $\frac{s+1}{(s-5)(s+1)}$
    Notice that the $(s+1)$ terms cancel out!
  3. Reduced Form:
    $\mathcal{L}^{-1}\left\{ \frac{1}{s-5} \right\}$
  4. Identify the Form: This matches the exponential rule $\frac{1}{s-a}$ where $a = 5$.
  5. Final Answer:
    $= e^{5t}$

Example 3: Inverse Translation

Evaluate: $\mathcal{L}^{-1}\left\{ \frac{2}{(s-3)^4} \right\}$

  1. Identify the Shift: The denominator has $(s-a)^4$. This means $a = 3$, which results in an $e^{3t}$ multiplier outside the transform.
  2. Un-shift the Transform:
    $= e^{3t} \mathcal{L}^{-1}\left\{ \frac{2}{s^4} \right\}$
  3. Match the Table: We need $\frac{n!}{s^{n+1}}$. Here $n+1 = 4$, so $n=3$. We need $3! = 6$ in the numerator.
  4. Multiply and Divide by 3!:
    $= e^{3t} \mathcal{L}^{-1}\left\{ \frac{2}{6} \cdot \frac{6}{s^4} \right\} = \frac{1}{3} e^{3t} \mathcal{L}^{-1}\left\{ \frac{3!}{s^{3+1}} \right\}$
  5. Final Answer:
    $= \frac{1}{3} t^3 e^{3t}$

Example 4: Completing the Square

Evaluate: $\mathcal{L}^{-1}\left\{ \frac{s}{s^2 + 4s + 13} \right\}$

If the denominator is an unfactorable quadratic, complete the square to find the $s$-shift!

  1. Complete the Square:
    $s^2 + 4s + 4 - 4 + 13 = (s+2)^2 + 9$. The shift is $a = -2$.
  2. Match the Shift in the Numerator: To apply the translation theorem, every $s$ must be $(s+2)$. Add and subtract $2$ on top.
    $\frac{s+2-2}{(s+2)^2 + 9}$
  3. Split the Fraction:
    $\mathcal{L}^{-1}\left\{ \frac{s+2}{(s+2)^2 + 9} \right\} - \mathcal{L}^{-1}\left\{ \frac{2}{(s+2)^2 + 9} \right\}$
  4. Pull out the Shift ($e^{-2t}$):
    $= e^{-2t} \mathcal{L}^{-1}\left\{ \frac{s}{s^2 + 9} \right\} - e^{-2t} \mathcal{L}^{-1}\left\{ \frac{2}{s^2 + 9} \right\}$
  5. Adjust Constants for Sine ($b=3$):
    $= e^{-2t} \cos(3t) - e^{-2t} \left(\frac{2}{3}\right) \mathcal{L}^{-1}\left\{ \frac{3}{s^2 + 9} \right\}$
  6. Final Answer:
    $= e^{-2t} \cos(3t) - \frac{2}{3} e^{-2t} \sin(3t)$

Example 5: Full Partial Fractions

Evaluate: $\mathcal{L}^{-1}\left\{ \frac{1}{s(s^2+1)} \right\}$

  1. Set up the Decomposition:
    $\frac{1}{s(s^2+1)} = \frac{A}{s} + \frac{Bs+C}{s^2+1}$
  2. Clear Denominators:
    $1 = A(s^2+1) + (Bs+C)s = As^2 + A + Bs^2 + Cs = (A+B)s^2 + Cs + A$
  3. Equate Coefficients:
    Constants: $A = 1$
    $s$ terms: $C = 0$
    $s^2$ terms: $A+B = 0 \implies 1+B=0 \implies B=-1$
  4. Rewrite the Inverse Transform:
    $\mathcal{L}^{-1}\left\{ \frac{1}{s} - \frac{s}{s^2+1} \right\}$
  5. Final Answer:
    $= 1 - \cos t$

Example 6: Solving an IVP

Solve the Initial Value Problem: $y'' + 4y = 0$, given $y(0)=2, y'(0)=2$

  1. Take Laplace of Both Sides: Let $Y(s) = \mathcal{L}\{y(t)\}$.
    $[s^2Y(s) - sy(0) - y'(0)] + 4Y(s) = 0$
  2. Substitute Initial Conditions:
    $s^2Y(s) - 2s - 2 + 4Y(s) = 0$
  3. Isolate $Y(s)$:
    $Y(s)(s^2 + 4) = 2s + 2 \implies Y(s) = \frac{2s + 2}{s^2 + 4}$
  4. Prepare for Inverse Transform: Split the fraction.
    $Y(s) = \frac{2s}{s^2 + 4} + \frac{2}{s^2 + 4}$
  5. Take the Inverse Laplace Transform:
    $y(t) = 2\mathcal{L}^{-1}\left\{ \frac{s}{s^2+4} \right\} + \mathcal{L}^{-1}\left\{ \frac{2}{s^2+4} \right\}$
  6. Final Answer:
    $y(t) = 2\cos(2t) + \sin(2t)$
