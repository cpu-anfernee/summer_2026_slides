The Laplace Transform

Definition, Linearity, and Translation Properties

The Definition

The Laplace transform is an integral transform that converts a function of time $f(t)$ into a function of a complex variable $s$. It translates differential equations into algebraic equations.

Let $f(t)$ be defined for $t \ge 0$. The Laplace Transform of $f(t)$ is:

\[ \mathcal{L}\{f(t)\} = \int_0^\infty e^{-st} f(t) dt = F(s) \]

Linearity Property

The Laplace transform is a linear operator. For constants $\alpha$ and $\beta$:

\[ \mathcal{L}\{\alpha f(t) + \beta g(t)\} = \alpha \mathcal{L}\{f(t)\} + \beta \mathcal{L}\{g(t)\} \]

Table of Common Transforms

These fundamental transforms are derived directly from the integral definition and should be memorized.

$\mathcal{L}\{1\} = \frac{1}{s}$

$\mathcal{L}\{t^n\} = \frac{n!}{s^{n+1}}$

$\mathcal{L}\{e^{at}\} = \frac{1}{s-a}$

$\mathcal{L}\{\sin bt\} = \frac{b}{s^2 + b^2}$

$\mathcal{L}\{\cos bt\} = \frac{s}{s^2 + b^2}$

First Translation (Shifting) Theorem

Multiplying by an exponential in the $t$-domain shifts the variable in the $s$-domain.

\[ \mathcal{L}\{e^{at} f(t)\} = F(s - a) \]

Example 1: Using the Definition

Evaluate $\mathcal{L}\{e^{at}\}$ using the integral definition.

  1. Set up the Integral:
    $\mathcal{L}\{e^{at}\} = \int_0^\infty e^{-st} e^{at} dt$
  2. Combine Exponents:
    $= \int_0^\infty e^{-(s-a)t} dt$
  3. Integrate (treat $s, a$ as constants):
    $= \lim_{b \to \infty} \left[ \frac{e^{-(s-a)t}}{-(s-a)} \right]_0^b$
  4. Evaluate the Limits: Assuming $s > a$, the upper limit goes to $0$.
    $= 0 - \left( \frac{e^0}{-(s-a)} \right)$
  5. Final Answer:
    $= \frac{1}{s-a}$

Example 2: Applying Linearity

Find the Laplace transform of $f(t) = 4t^2 - 3\cos(5t) + 2e^{-t}$.

  1. Distribute the Operator: Use the linearity property.
    $\mathcal{L}\{f(t)\} = 4\mathcal{L}\{t^2\} - 3\mathcal{L}\{\cos(5t)\} + 2\mathcal{L}\{e^{-t}\}$
  2. Apply Basic Formulas:
    $\mathcal{L}\{t^2\} = \frac{2!}{s^{2+1}} = \frac{2}{s^3}$
    $\mathcal{L}\{\cos(5t)\} = \frac{s}{s^2 + 5^2} = \frac{s}{s^2 + 25}$
    $\mathcal{L}\{e^{-t}\} = \frac{1}{s - (-1)} = \frac{1}{s + 1}$
  3. Substitute Back:
    $= 4\left(\frac{2}{s^3}\right) - 3\left(\frac{s}{s^2 + 25}\right) + 2\left(\frac{1}{s + 1}\right)$
  4. Final Answer:
    $F(s) = \frac{8}{s^3} - \frac{3s}{s^2 + 25} + \frac{2}{s + 1}$

Example 3: Trigonometric Identities

Find $\mathcal{L}\{\sin^2 t\}$.

There is no rule for products or powers! We must use the half-angle identity.

  1. Rewrite the Function:
    $\sin^2 t = \frac{1 - \cos(2t)}{2} = \frac{1}{2} - \frac{1}{2}\cos(2t)$
  2. Distribute the Operator:
    $\mathcal{L}\{\sin^2 t\} = \frac{1}{2}\mathcal{L}\{1\} - \frac{1}{2}\mathcal{L}\{\cos(2t)\}$
  3. Apply Basic Formulas:
    $\mathcal{L}\{1\} = \frac{1}{s}$
    $\mathcal{L}\{\cos(2t)\} = \frac{s}{s^2 + 4}$
  4. Substitute Back:
    $= \frac{1}{2}\left(\frac{1}{s}\right) - \frac{1}{2}\left(\frac{s}{s^2 + 4}\right)$
  5. Final Answer:
    $F(s) = \frac{1}{2s} - \frac{s}{2(s^2 + 4)}$

Example 4: First Translation Theorem

Find $\mathcal{L}\{e^{2t} \sin(3t)\}$.

  1. Identify $f(t)$ and $a$: The exponential $e^{2t}$ shifts the transform of the remaining function. Here, $a = 2$ and $f(t) = \sin(3t)$.
  2. Find the Transform of $f(t)$:
    $F(s) = \mathcal{L}\{\sin(3t)\} = \frac{3}{s^2 + 3^2} = \frac{3}{s^2 + 9}$
  3. Apply the Shift: Evaluate $F(s)$ at $(s - a)$, which is $(s - 2)$.
    Replace every $s$ in the result with $(s - 2)$.
  4. Final Answer:
    $\mathcal{L}\{e^{2t} \sin(3t)\} = \frac{3}{(s - 2)^2 + 9}$

Example 5: Piecewise Function

Find the transform of $f(t) = \begin{cases} 3, & 0 \le t < 2 \\ 0, & t \ge 2 \end{cases}$

  1. Set up the Definition: Break the integral at $t = 2$.
    $\mathcal{L}\{f(t)\} = \int_0^2 e^{-st}(3) dt + \int_2^\infty e^{-st}(0) dt$
  2. Simplify: The second integral is 0.
    $= 3 \int_0^2 e^{-st} dt$
  3. Integrate:
    $= 3 \left[ \frac{e^{-st}}{-s} \right]_0^2 = -\frac{3}{s} \left[ e^{-st} \right]_0^2$
  4. Evaluate at Limits:
    $= -\frac{3}{s} (e^{-2s} - e^0)$
  5. Final Answer:
    $F(s) = \frac{3}{s} (1 - e^{-2s})$

Example 6: Transform of a Derivative

Given the rule $\mathcal{L}\{y''\} = s^2Y(s) - sy(0) - y'(0)$, expand the transform of the differential equation: $y'' + 4y = e^{-t}$.

  1. Take Laplace of Both Sides: Let $Y(s) = \mathcal{L}\{y(t)\}$.
    $\mathcal{L}\{y''\} + 4\mathcal{L}\{y\} = \mathcal{L}\{e^{-t}\}$
  2. Expand the Left Side:
    $[s^2 Y(s) - sy(0) - y'(0)] + 4Y(s)$
  3. Evaluate the Right Side:
    $\mathcal{L}\{e^{-t}\} = \frac{1}{s + 1}$
  4. Equate and Group $Y(s)$ terms:
    $(s^2 + 4)Y(s) - sy(0) - y'(0) = \frac{1}{s + 1}$
  5. Conclusion:
    This algebraic equation can now be solved for $Y(s)$ simply by substituting the initial conditions and isolating $Y(s)$!
