Chapter 4.2

Area Under a Curve

Using Definite Integrals to Calculate Area

The Definition of Area

If a function \( f \) is continuous and nonnegative (\( f(x) \ge 0 \)) on the closed interval \( [a, b] \), then the area of the region bounded by the graph of \( f \), the \( x \)-axis, and the vertical lines \( x = a \) and \( x = b \) is given by:

\[ \text{Area} = \int_a^b f(x) dx \]

What if the curve is below the x-axis?

If \( f(x) \le 0 \) on \( [a, b] \), the definite integral will yield a negative number. Since geometric area must be positive, the actual area is the absolute value of the integral:

\[ \text{Area} = \left| \int_a^b f(x) dx \right| = -\int_a^b f(x) dx \]

Example 1: Basic Polynomial Area

Find the area of the region bounded by \( y = x^2 \), the x-axis, and the vertical lines \( x = 0 \) and \( x = 2 \).

  1. Set up the Integral: The function is nonnegative on \( [0, 2] \).
    \( \text{Area} = \int_0^2 x^2 dx \)
  2. Find the Antiderivative: Use the Power Rule.
    \( \int x^2 dx = \frac{x^3}{3} \)
  3. Apply the Fundamental Theorem of Calculus: Evaluate from 0 to 2.
    \( \left[ \frac{x^3}{3} \right]_0^2 = \frac{2^3}{3} - \frac{0^3}{3} \)
  4. Final Answer:
    \( \text{Area} = \frac{8}{3} \)

Example 2: Verifying Geometry

Find the area bounded by \( y = 2x \), the x-axis, \( x = 0 \), and \( x = 3 \).

  1. Geometric Check: This region is a right triangle with base \( 3 \) and height \( f(3) = 6 \).
    \( \text{Geometric Area} = \frac{1}{2} b h = \frac{1}{2}(3)(6) = 9 \). Let's verify with calculus.
  2. Set up the Integral:
    \( \text{Area} = \int_0^3 2x dx \)
  3. Integrate:
    \( \left[ x^2 \right]_0^3 \)
  4. Evaluate:
    \( 3^2 - 0^2 = 9 \). The calculus matches the geometry!

Example 3: Area Below the x-axis

Find the area of the region bounded by \( y = x^3 \), the x-axis, \( x = -2 \), and \( x = 0 \).

  1. Analyze the Region: On the interval \( [-2, 0] \), the function \( y = x^3 \) is entirely below the x-axis. The integral will be negative.
  2. Set up the Integral:
    \( \int_{-2}^0 x^3 dx \)
  3. Integrate:
    \( \left[ \frac{x^4}{4} \right]_{-2}^0 \)
  4. Evaluate:
    \( \frac{0^4}{4} - \frac{(-2)^4}{4} = 0 - \frac{16}{4} = -4 \)
  5. Find the Area: Area must be positive.
    \( \text{Area} = |-4| = 4 \)

Example 4: Crossing the x-axis (Total Area)

Find the total area bounded by \( y = x^2 - 4 \), the x-axis, \( x = 0 \), and \( x = 3 \).

Warning: The curve crosses the x-axis at \( x=2 \). We must split the integral!

  1. Set up the Split Integrals: Part 1 (below x-axis) is \( \). Part 2 (above) is \( \).
    \( \text{Total Area} = \left| \int_0^2 (x^2 - 4) dx \right| + \int_2^3 (x^2 - 4) dx \)
  2. Evaluate Part 1:
    \( \left[ \frac{x^3}{3} - 4x \right]_0^2 = \left( \frac{8}{3} - 8 \right) - 0 = -\frac{16}{3} \).
    Area of this piece is \( \left|-\frac{16}{3}\right| = \frac{16}{3} \).
  3. Evaluate Part 2:
    \( \left[ \frac{x^3}{3} - 4x \right]_2^3 = \left( \frac{27}{3} - 12 \right) - \left( -\frac{16}{3} \right) = -3 + \frac{16}{3} = \frac{7}{3} \).
  4. Final Total Area:
    \( \text{Total Area} = \frac{16}{3} + \frac{7}{3} = \frac{23}{3} \)

Example 5: Trigonometric Area

Find the area of the region bounded by \( y = \sin x \) and the x-axis on the interval \( [0, \pi] \).

  1. Analyze the Region: The sine wave is positive from \( 0 \) to \( \pi \), so we can integrate directly.
  2. Set up the Integral:
    \( \text{Area} = \int_0^\pi \sin x dx \)
  3. Find the Antiderivative:
    \( \left[ -\cos x \right]_0^\pi \)
  4. Evaluate:
    \( (-\cos \pi) - (-\cos 0) \)
  5. Final Answer:
    \( -(-1) - (-1) = 1 + 1 = 2 \)

Example 6: Radical Function Area

Find the area of the region bounded by \( y = \sqrt{x} \), the x-axis, \( x = 1 \), and \( x = 4 \).

  1. Set up the Integral: The square root function is positive on this interval.
    \( \text{Area} = \int_1^4 x^{1/2} dx \)
  2. Integrate: Apply the Power Rule.
    \( \left[ \frac{x^{3/2}}{3/2} \right]_1^4 = \left[ \frac{2}{3}x^{3/2} \right]_1^4 \)
  3. Evaluate the Limits:
    Upper limit: \( \frac{2}{3}(4^{3/2}) = \frac{2}{3}(\sqrt{4})^3 = \frac{2}{3}(2)^3 = \frac{16}{3} \).
    Lower limit: \( \frac{2}{3}(1^{3/2}) = \frac{2}{3}(1) = \frac{2}{3} \).
  4. Final Answer:
    \( \text{Area} = \frac{16}{3} - \frac{2}{3} = \frac{14}{3} \)
