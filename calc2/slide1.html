Chapter 4.1

Antiderivatives and Indefinite Integration

Part 1: Basic Rules and Algebraic Methods

What is an Antiderivative?

Differentiation is the process of finding a derivative. Antidifferentiation is the reverse: finding the original function when you are given the derivative.

A function \(F\) is an antiderivative of \(f\) on an interval \(I\) when \(F^{\prime}(x)=f(x)\) for all \(x\) in \(I\).

The Constant of Integration (+C)

Because the derivative of any constant is zero, a function has an infinite number of antiderivatives.

For example, if \(F(x) = x^2\), then \(F'(x) = 2x\).
But if \(G(x) = x^2 + 5\), then \(G'(x) = 2x\) as well.

Representation of Antiderivatives (Theorem 4.1):

If \(F\) is an antiderivative of \(f\) on an interval \(I\), then \(G\) is an antiderivative of \(f\) on the interval \(I\) if and only if \(G\) is of the form \(G(x)=F(x)+C\) for all \(x\) in \(I\), where \(C\) is a constant.

Indefinite Integral Notation

The operation of finding all solutions of this equation is called antidifferentiation (or indefinite integration) and is denoted by an integral sign \(\int\).

\[ y=\int f(x)dx=F(x)+C \]
  • \(\int\) : The integral sign indicating the operation.
  • \(f(x)\) : The Integrand (the function being integrated).
  • \(dx\) : Identifies \(x\) as the variable of integration.
  • \(C\) : The constant of integration.

Basic Integration Rules

The inverse nature of integration and differentiation means we can derive integration rules directly from differentiation rules.

  • Constant Rule: \(\int k~dx=kx+C\)
  • Constant Multiple Rule: \(\int kf(x)dx=k\int f(x)dx\)
  • Sum & Difference Rule: \(\int [f(x)\pm g(x)]dx=\int f(x)dx\pm\int g(x)dx\)

The Power Rule:

\[ \int x^{n}dx=\frac{x^{n+1}}{n+1}+C, \quad (n \neq -1) \]

(Note: We add 1 to the power, then divide by the new power.)

Step-by-Step Example 1

Evaluate the indefinite integral: \(\int 3x~dx\)

  1. Apply the Constant Multiple Rule: Factor the 3 outside the integral.
    \(= 3\int x^{1}dx\)
  2. Apply the Power Rule (\(n=1\)): Add 1 to the exponent, then divide by the new exponent.
    \(= 3\left(\frac{x^{1+1}}{1+1}\right) + C\)
  3. Simplify:
    \(= 3\left(\frac{x^{2}}{2}\right) + C\)

    Final Answer: \(= \frac{3}{2}x^{2} + C\)

Step-by-Step Example 2

Evaluate the indefinite integral: \(\int \frac{1}{x^{3}}dx\)

  1. Rewrite the Integrand: Bring \(x\) to the numerator by making the exponent negative. This prepares it for the Power Rule.
    \(= \int x^{-3}dx\)
  2. Apply the Power Rule (\(n=-3\)): Add 1 to the exponent (\(-3 + 1 = -2\)) and divide by \(-2\).
    \(= \frac{x^{-2}}{-2} + C\)
  3. Simplify: Move the negative exponent back to the denominator for a cleaner final answer.

    Final Answer: \(= -\frac{1}{2x^{2}} + C\)

Step-by-Step Example 3

Evaluate the indefinite integral: \(\int \sqrt{x}dx\)

  1. Rewrite the Integrand: Convert the radical into a fractional exponent.
    \(= \int x^{1/2}dx\)
  2. Apply the Power Rule (\(n=1/2\)): Add 1 to the exponent (\(1/2 + 2/2 = 3/2\)) and divide by \(3/2\).
    \(= \frac{x^{3/2}}{3/2} + C\)
  3. Simplify: Dividing by a fraction is the same as multiplying by its reciprocal.

    Final Answer: \(= \frac{2}{3}x^{3/2} + C\)

Step-by-Step Example 4

Evaluate: \(\int (3x^{4}-5x^{2}+x)dx\)

  1. Apply the Sum/Difference Rule: Integrate each term separately.
    \(= \int 3x^{4}dx - \int 5x^{2}dx + \int x dx\)
  2. Apply the Constant Multiple & Power Rules:
    \(= 3\left(\frac{x^{5}}{5}\right) - 5\left(\frac{x^{3}}{3}\right) + \frac{x^{2}}{2} + C\)
    (Note: We only need to write '+ C' once at the very end).
  3. Simplify: Clean up the coefficients.

    Final Answer: \(= \frac{3}{5}x^{5} - \frac{5}{3}x^{3} + \frac{1}{2}x^{2} + C\)

Step-by-Step Example 5

Evaluate: \(\int \frac{x+1}{\sqrt{x}}dx\)

Warning: There is no "Quotient Rule" for integration. You cannot integrate the top and bottom separately!

  1. Rewrite by Splitting the Fraction: Divide each term in the numerator by the denominator (\(\sqrt{x}\) is \(x^{1/2}\)).
    \(= \int \left(\frac{x}{x^{1/2}} + \frac{1}{x^{1/2}}\right)dx\)
  2. Simplify Exponents: Use exponent rules (subtracting powers for division).
    \(= \int \left(x^{1/2} + x^{-1/2}\right)dx\)
  3. Integrate using the Power Rule: Add 1 to each exponent and divide.
    \(= \frac{x^{3/2}}{3/2} + \frac{x^{1/2}}{1/2} + C\)
  4. Simplify: Multiply by the reciprocals.

    Final Answer: \(= \frac{2}{3}x^{3/2} + 2x^{1/2} + C\)
