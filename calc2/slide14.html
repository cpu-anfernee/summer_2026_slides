Chapter 8.5

Partial Fractions

Integration of Rational Functions

The Method of Partial Fractions

Partial fraction decomposition allows us to break down complex rational functions $f(x) = \frac{P(x)}{Q(x)}$ into a sum of simpler fractions that are easy to integrate.

Step 1: Proper vs. Improper

If the degree of the numerator $P(x)$ is greater than or equal to the degree of the denominator $Q(x)$, the fraction is improper. You must use polynomial long division first!

\[ \frac{P(x)}{Q(x)} = \text{Quotient} + \frac{\text{Remainder}}{Q(x)} \]

The Four Denominator Cases

Factor the denominator $Q(x)$ completely. The factors dictate the form of the decomposition.

Case 1: Distinct Linear Factors

For each factor $(ax+b)$, add a term: $\frac{A}{ax+b}$

Case 2: Repeated Linear Factors

For $(ax+b)^k$, add $k$ terms: $\frac{A_1}{ax+b} + \frac{A_2}{(ax+b)^2} + \dots + \frac{A_k}{(ax+b)^k}$

Case 3: Distinct Irreducible Quadratic Factors

For each factor $(ax^2+bx+c)$, add a term: $\frac{Ax+B}{ax^2+bx+c}$

Case 4: Repeated Irreducible Quadratic Factors

For $(ax^2+bx+c)^k$, add $k$ terms: $\frac{A_1x+B_1}{ax^2+bx+c} + \frac{A_2x+B_2}{(ax^2+bx+c)^2} + \dots$

Example 1: Distinct Linear Factors

Evaluate: $\int \frac{1}{x^2 - 5x + 6} dx$

  1. Factor the Denominator:
    $x^2 - 5x + 6 = (x - 2)(x - 3)$.
  2. Set up the Decomposition:
    $\frac{1}{(x - 2)(x - 3)} = \frac{A}{x - 2} + \frac{B}{x - 3}$
  3. Clear Denominators (Basic Equation):
    $1 = A(x - 3) + B(x - 2)$
  4. Solve for A and B (Heaviside Cover-Up):
    Let $x = 3 \implies 1 = A(0) + B(1) \implies B = 1$
    Let $x = 2 \implies 1 = A(-1) + B(0) \implies A = -1$
  5. Rewrite and Integrate:
    $\int \left( \frac{-1}{x - 2} + \frac{1}{x - 3} \right) dx$
  6. Final Answer:
    $-\ln|x - 2| + \ln|x - 3| + C = \ln\left|\frac{x - 3}{x - 2}\right| + C$

Example 2: Repeated Linear Factors

Evaluate: $\int \frac{5x^2 + 20x + 6}{x(x + 1)^2} dx$

  1. Set up the Decomposition: Since $(x+1)$ is squared, it gets two terms.
    $\frac{5x^2 + 20x + 6}{x(x + 1)^2} = \frac{A}{x} + \frac{B}{x + 1} + \frac{C}{(x + 1)^2}$
  2. Clear Denominators:
    $5x^2 + 20x + 6 = A(x + 1)^2 + Bx(x + 1) + Cx$
  3. Solve for Constants:
    Let $x = 0 \implies 6 = A(1)^2 \implies A = 6$
    Let $x = -1 \implies 5(-1)^2 + 20(-1) + 6 = C(-1) \implies -9 = -C \implies C = 9$
    To find $B$, pick any other $x$, say $x = 1$:
    $5(1) + 20(1) + 6 = 6(2)^2 + B(1)(2) + 9(1) \implies 31 = 24 + 2B + 9 \implies 2B = -2 \implies B = -1$
  4. Rewrite and Integrate:
    $\int \left( \frac{6}{x} - \frac{1}{x + 1} + \frac{9}{(x + 1)^2} \right) dx$
  5. Final Answer: Note the Power Rule for the third term!
    $6\ln|x| - \ln|x + 1| - \frac{9}{x + 1} + C$

Example 3: Irreducible Quadratic

Evaluate: $\int \frac{2x^2 - x + 4}{x(x^2 + 4)} dx$

  1. Set up the Decomposition: $x^2+4$ cannot be factored.
    $\frac{2x^2 - x + 4}{x(x^2 + 4)} = \frac{A}{x} + \frac{Bx + C}{x^2 + 4}$
  2. Clear Denominators:
    $2x^2 - x + 4 = A(x^2 + 4) + (Bx + C)x = (A + B)x^2 + Cx + 4A$
  3. Equate Coefficients:
    Constants: $4 = 4A \implies A = 1$
    $x$ terms: $-1 = C \implies C = -1$
    $x^2$ terms: $2 = A + B \implies 2 = 1 + B \implies B = 1$
  4. Rewrite the Integral:
    $\int \left( \frac{1}{x} + \frac{x - 1}{x^2 + 4} \right) dx = \int \frac{1}{x} dx + \int \frac{x}{x^2 + 4} dx - \int \frac{1}{x^2 + 4} dx$
  5. Final Answer: Use u-sub for the middle term, arctan for the last term.
    $\ln|x| + \frac{1}{2}\ln(x^2 + 4) - \frac{1}{2}\arctan\left(\frac{x}{2}\right) + C$

Example 4: Improper Rational Function

Evaluate: $\int \frac{x^3 + x}{x - 1} dx$

The degree of the numerator (3) is greater than the denominator (1). We must divide first!

  1. Polynomial Long Division: Divide $(x^3 + x)$ by $(x - 1)$.
    Quotient: $x^2 + x + 2$
    Remainder: $2$
  2. Rewrite the Integrand:
    $\frac{x^3 + x}{x - 1} = x^2 + x + 2 + \frac{2}{x - 1}$
  3. Set up the New Integral:
    $\int \left( x^2 + x + 2 + \frac{2}{x - 1} \right) dx$
  4. Integrate Term by Term:
    The first three terms use the Power Rule. The last term is a natural logarithm.
  5. Final Answer:
    $\frac{x^3}{3} + \frac{x^2}{2} + 2x + 2\ln|x - 1| + C$

Example 5: Deriving a Standard Formula

Evaluate: $\int \frac{dx}{x^2 - a^2}$

  1. Factor the Denominator:
    $x^2 - a^2 = (x - a)(x + a)$.
  2. Set up the Decomposition:
    $\frac{1}{(x - a)(x + a)} = \frac{A}{x - a} + \frac{B}{x + a}$
  3. Clear Denominators:
    $1 = A(x + a) + B(x - a)$
  4. Solve for A and B:
    Let $x = a \implies 1 = A(2a) \implies A = \frac{1}{2a}$
    Let $x = -a \implies 1 = B(-2a) \implies B = -\frac{1}{2a}$
  5. Rewrite and Integrate:
    $\frac{1}{2a} \int \left( \frac{1}{x - a} - \frac{1}{x + a} \right) dx$
  6. Final Answer:
    $\frac{1}{2a} \ln|x - a| - \frac{1}{2a} \ln|x + a| + C = \frac{1}{2a} \ln\left|\frac{x - a}{x + a}\right| + C$

Example 6: A Complex Setup

Evaluate: $\int \frac{x^2 + 1}{(x - 3)(x - 2)^2} dx$

  1. Set up the Decomposition:
    $\frac{x^2 + 1}{(x - 3)(x - 2)^2} = \frac{A}{x - 3} + \frac{B}{x - 2} + \frac{C}{(x - 2)^2}$
  2. Clear Denominators:
    $x^2 + 1 = A(x - 2)^2 + B(x - 3)(x - 2) + C(x - 3)$
  3. Solve for Constants:
    Let $x = 3 \implies 10 = A(1)^2 \implies A = 10$
    Let $x = 2 \implies 5 = C(-1) \implies C = -5$
    Let $x = 0 \implies 1 = 10(4) + B(-3)(-2) - 5(-3) \implies 1 = 40 + 6B + 15 \implies 6B = -54 \implies B = -9$
  4. Rewrite the Integral:
    $\int \left( \frac{10}{x - 3} - \frac{9}{x - 2} - \frac{5}{(x - 2)^2} \right) dx$
  5. Final Answer:
    $10\ln|x - 3| - 9\ln|x - 2| + \frac{5}{x - 2} + C$
