Chapter 5.2

The Natural Logarithmic Function

Integration Using the Log Rule

The Log Rule for Integration

In Chapter 4, we learned the Power Rule for integration: \( \int x^n dx = \frac{x^{n+1}}{n+1} + C \). However, this rule fails when \( n = -1 \).

The Log Rule answers this missing case:

\[ \int \frac{1}{x} dx = \ln|x| + C \]

Note the absolute value bars! They are required because the domain of the natural logarithm function does not include negative numbers, but the domain of \( \frac{1}{x} \) does.

The General Log Rule

When combined with substitution (the Chain Rule in reverse), the Log Rule takes a powerful generalized form.

If \( u \) is a differentiable function of \( x \):

\[ \int \frac{1}{u} du = \ln|u| + C \]

Alternatively written as recognizing the derivative of the denominator in the numerator:

\[ \int \frac{u^{\prime}}{u} dx = \ln|u| + C \]

Examples of Logarithmic Integration

The following slides contain step-by-step solutions to integrals that yield natural logarithms.

Example 1: Basic Constant Multiple

Evaluate: \( \int \frac{5}{x} dx \)

  1. Factor the Constant: Pull the 5 outside the integral.
    \( = 5 \int \frac{1}{x} dx \)
  2. Apply the Log Rule: Recognize the standard form.
    \( = 5 \ln|x| + C \)
  3. Final Answer:
    \( 5 \ln|x| + C \)

Example 2: Simple u-Substitution

Evaluate: \( \int \frac{1}{x - 4} dx \)

  1. Set up \( u \): Let the denominator be \( u \).
    \( u = x - 4 \)
  2. Find \( du \): The derivative of \( x-4 \) is 1.
    \( du = dx \)
  3. Substitute and Integrate:
    \( = \int \frac{1}{u} du = \ln|u| + C \)
  4. Substitute Back: Replace \( u \) with \( x-4 \).
    Final Answer: \( \ln|x - 4| + C \)

Example 3: Adjusting the Numerator

Evaluate: \( \int \frac{x}{x^2 + 1} dx \)

  1. Set up \( u \) and \( du \):
    Let \( u = x^2 + 1 \). Then \( du = 2x dx \).
  2. Adjust Constants: We have an \( x \) in the numerator, but we need \( 2x \). Multiply inside by 2 and outside by 1/2.
    \( = \frac{1}{2} \int \frac{2x}{x^2 + 1} dx \)
  3. Substitute and Integrate: Now we have \( \frac{1}{2} \int \frac{1}{u} du \).
    \( = \frac{1}{2} \ln|u| + C \)
  4. Final Answer:
    \( = \frac{1}{2} \ln(x^2 + 1) + C \)
    (Note: Absolute value bars can be dropped here since \( x^2+1 \) is always positive).

Example 4: Direct Pattern Recognition

Evaluate: \( \int \frac{3x^2 + 4}{x^3 + 4x - 3} dx \)

Look for the \( \frac{u^{\prime}}{u} \) pattern directly!

  1. Identify the Denominator's Derivative:
    Let \( u = x^3 + 4x - 3 \).
    \( u^{\prime} = 3x^2 + 4 \)
  2. Verify: The numerator is exactly the derivative of the denominator!
  3. Apply the General Rule Immediately:
    Final Answer: \( \ln|x^3 + 4x - 3| + C \)

Example 5: Improper Rational Functions

Evaluate: \( \int \frac{x^2 + x + 1}{x^2 + 1} dx \)

  1. Recognize the Problem: The degree of the numerator (2) is equal to or greater than the denominator (2). We must use long division or algebraic manipulation first.
  2. Rewrite the Fraction: Split the numerator to group \( x^2+1 \) together.
    \( \frac{x^2 + 1 + x}{x^2 + 1} = \frac{x^2 + 1}{x^2 + 1} + \frac{x}{x^2 + 1} = 1 + \frac{x}{x^2 + 1} \)
  3. Set up the New Integral:
    \( \int \left( 1 + \frac{x}{x^2 + 1} \right) dx = \int 1 dx + \int \frac{x}{x^2 + 1} dx \)
  4. Integrate Separately: We solved the right side in Example 3.
    Left side: \( x \)
    Right side: \( \frac{1}{2} \ln(x^2 + 1) \)
  5. Final Answer:
    \( x + \frac{1}{2} \ln(x^2 + 1) + C \)

Example 6: Trigonometric Rewriting

Evaluate: \( \int \tan x \, dx \)

Wait, there's no fraction here! Let's make one using identities.

  1. Rewrite as Sine/Cosine:
    \( \int \frac{\sin x}{\cos x} dx \)
  2. Set up \( u \): Let \( u \) be the denominator.
    \( u = \cos x \)
  3. Find \( du \):
    \( du = -\sin x \, dx \implies -du = \sin x \, dx \)
  4. Substitute and Integrate:
    \( \int \frac{1}{u} (-du) = -\int \frac{1}{u} du = -\ln|u| + C \)
  5. Final Answer:
    \( -\ln|\cos x| + C \)
    (Note: By logarithmic properties, this is also equal to \( \ln|\sec x| + C \)).
