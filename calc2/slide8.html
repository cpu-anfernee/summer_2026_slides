Chapters 4.3 & 4.5

Definite Integrals

Properties, Symmetry, Change of Variables, & Absolute Values

1. Properties of Definite Integrals (Ch 4.3)

These properties allow us to manipulate and break down complex integrals.

  1. Zero Interval: \( \int_a^a f(x) dx = 0 \)
  2. Reversing Limits: \( \int_b^a f(x) dx = -\int_a^b f(x) dx \)
  3. Additive Interval: \( \int_a^b f(x) dx + \int_b^c f(x) dx = \int_a^c f(x) dx \)
  4. Constant Multiple: \( \int_a^b k f(x) dx = k \int_a^b f(x) dx \)
  5. Sum/Difference: \( \int_a^b [f(x) \pm g(x)] dx = \int_a^b f(x) dx \pm \int_a^b g(x) dx \)

2. Even and Odd Functions (Ch 4.5)

When integrating over a symmetric interval \( [-a, a] \), symmetry provides massive shortcuts.

Even Functions (\( y \)-axis symmetry): \( f(-x) = f(x) \)

\[ \int_{-a}^a f(x) dx = 2 \int_0^a f(x) dx \]

Odd Functions (Origin symmetry): \( f(-x) = -f(x) \)

\[ \int_{-a}^a f(x) dx = 0 \]

3. Change of Variables (Ch 4.5)

When using \( u \)-substitution on a definite integral, you must change the limits of integration to match the new variable \( u \). This prevents the need to substitute \( x \) back in at the end!

If \( u = g(x) \), then:

\[ \int_a^b f(g(x))g^{\prime}(x) dx = \int_{g(a)}^{g(b)} f(u) du \]
  • New lower limit: Plug \( a \) into \( u \).
  • New upper limit: Plug \( b \) into \( u \).

4. Absolute Value Functions

To integrate an absolute value function, you must split the integral at its "vertex" (the zero-point) using the Additive Interval property.

By definition, \( |x| = \begin{cases} -x & \text{for } x < 0 \\ x & \text{for } x \ge 0 \end{cases} \)


Example: \( \int_a^b |x - c| dx \) (where \( a < c < b \))

\[ = \int_a^c -(x - c) dx + \int_c^b (x - c) dx \]

Example 1: Using Properties

Given \( \int_1^3 f(x) dx = 4 \) and \( \int_1^3 g(x) dx = -2 \).

Evaluate: \( \int_1^3 [3f(x) - 2g(x)] dx \)

  1. Apply Sum/Difference Rule: Separate the integral.
    \( \int_1^3 3f(x) dx - \int_1^3 2g(x) dx \)
  2. Apply Constant Multiple Rule: Pull constants outside.
    \( 3\int_1^3 f(x) dx - 2\int_1^3 g(x) dx \)
  3. Substitute Given Values:
    \( 3(4) - 2(-2) \)
  4. Simplify:
    \( 12 - (-4) \)
  5. Final Answer:
    \( 16 \)

Example 2: Odd Function Symmetry

Evaluate: \( \int_{-2}^2 (x^3 - \sin x) dx \)

  1. Check the Interval: The interval \( [-2, 2] \) is symmetric about the origin. Symmetry rules might apply!
  2. Test the Function for Symmetry: Let \( f(x) = x^3 - \sin x \).
    Find \( f(-x) \):
    \( f(-x) = (-x)^3 - \sin(-x) = -x^3 - (-\sin x) = -x^3 + \sin x \)
  3. Compare:
    \( -x^3 + \sin x = -(x^3 - \sin x) = -f(x) \).
    Because \( f(-x) = -f(x) \), the function is strictly odd.
  4. Apply the Theorem: The integral of an odd function over a symmetric interval is exactly zero. The positive area perfectly cancels the negative area.
  5. Final Answer:
    \( 0 \)

Example 3: Even Function Symmetry

Evaluate: \( \int_{-\pi/4}^{\pi/4} \cos(x) dx \)

  1. Check the Interval: The interval is symmetric (\( -a \) to \( a \)).
  2. Test the Function: Let \( f(x) = \cos(x) \).
    \( f(-x) = \cos(-x) = \cos(x) = f(x) \).
    The function is even.
  3. Apply the Theorem: We can evaluate from \( 0 \) to \( a \) and double it.
    \( 2 \int_0^{\pi/4} \cos(x) dx \)
  4. Integrate:
    \( = 2 [\sin(x)]_0^{\pi/4} \)
  5. Final Answer:
    \( 2 \left( \sin\left(\frac{\pi}{4}\right) - \sin(0) \right) = 2 \left( \frac{\sqrt{2}}{2} - 0 \right) = \sqrt{2} \)

Example 4: Change of Variables

Evaluate: \( \int_0^1 x(x^2 + 1)^3 dx \)

  1. Set up \( u \): Let \( u = x^2 + 1 \), so \( du = 2x dx \implies \frac{du}{2} = x dx \).
  2. Change the Limits:
    Lower limit: When \( x = 0 \), \( u = (0)^2 + 1 = 1 \).
    Upper limit: When \( x = 1 \), \( u = (1)^2 + 1 = 2 \).
  3. Rewrite the Entire Integral in terms of \( u \):
    \( \int_1^2 u^3 \left(\frac{du}{2}\right) = \frac{1}{2} \int_1^2 u^3 du \)
  4. Integrate:
    \( \frac{1}{2} \left[ \frac{u^4}{4} \right]_1^2 = \frac{1}{8} [u^4]_1^2 \)
  5. Evaluate (No back-substitution needed!):
    \( \frac{1}{8} (2^4 - 1^4) = \frac{1}{8}(16 - 1) = \frac{15}{8} \)

Example 5: Change of Limits (Logarithmic)

Evaluate: \( \int_1^e \frac{(\ln x)^2}{x} dx \)

  1. Set up \( u \): Let \( u = \ln x \), so \( du = \frac{1}{x} dx \).
  2. Change the Limits:
    Lower limit: When \( x = 1 \), \( u = \ln(1) = 0 \).
    Upper limit: When \( x = e \), \( u = \ln(e) = 1 \).
  3. Rewrite the Integral in terms of \( u \):
    \( \int_0^1 u^2 du \)
  4. Integrate:
    \( \left[ \frac{u^3}{3} \right]_0^1 \)
  5. Evaluate directly:
    \( \frac{1^3}{3} - \frac{0^3}{3} = \frac{1}{3} \)

Example 6: Absolute Value

Evaluate: \( \int_0^4 |x - 3| dx \)

Find where the inside of the absolute value is zero to split the integral!

  1. Find the Vertex (Zero-point): \( x - 3 = 0 \implies x = 3 \).
    Since \( 3 \) is between \( 0 \) and \( 4 \), we split the integral at \( x=3 \).
  2. Apply the Piecewise Definition:
    For \( x < 3 \), \( |x - 3| = -(x - 3) = -x + 3 \).
    For \( x > 3 \), \( |x - 3| = (x - 3) = x - 3 \).
  3. Set up the New Integrals:
    \( \int_0^3 (-x + 3) dx + \int_3^4 (x - 3) dx \)
  4. Integrate each piece:
    \( \left[ -\frac{x^2}{2} + 3x \right]_0^3 + \left[ \frac{x^2}{2} - 3x \right]_3^4 \)
  5. Evaluate and Add:
    Left Piece: \( \left(-\frac{9}{2} + 9\right) - 0 = \frac{9}{2} \)
    Right Piece: \( \left(\frac{16}{2} - 12\right) - \left(\frac{9}{2} - 9\right) = (8 - 12) - \left(-\frac{9}{2}\right) = -4 + 4.5 = \frac{1}{2} \)
    Final Answer: \( \frac{9}{2} + \frac{1}{2} = \frac{10}{2} = 5 \)
