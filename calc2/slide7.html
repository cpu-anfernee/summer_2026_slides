Chapter 5.9

Hyperbolic Functions

Integration Rules and Techniques

Standard Hyperbolic Integrals

The integration rules for hyperbolic functions mirror standard trigonometric integrals, but with fewer negative signs. Let \( u \) be a differentiable function of \( x \):

\( \int \sinh u \, du = \cosh u + C \)

\( \int \cosh u \, du = \sinh u + C \)

\( \int \text{sech}^2 u \, du = \tanh u + C \)

\( \int \text{csch}^2 u \, du = -\coth u + C \)

\( \int \text{sech} u \tanh u \, du = -\text{sech} u + C \)

\( \int \text{csch} u \coth u \, du = -\text{csch} u + C \)

Notice: Integrating sinh yields POSITIVE cosh, unlike standard trigonometry where integrating sine yields negative cosine!

Example 1: Linear Substitution

Evaluate: \( \int \cosh(4x) \, dx \)

  1. Set up \( u \): Let the inner argument be \( u \).
    \( u = 4x \)
  2. Find \( du \):
    \( du = 4 \, dx \implies dx = \frac{du}{4} \)
  3. Substitute and Integrate:
    \( \int \cosh(u) \left(\frac{du}{4}\right) = \frac{1}{4} \int \cosh(u) \, du \)
  4. Apply the Rule:
    \( = \frac{1}{4} \sinh(u) + C \)
  5. Final Answer:
    \( \frac{1}{4} \sinh(4x) + C \)

Example 2: Polynomial Argument

Evaluate: \( \int x \sinh(x^2) \, dx \)

  1. Set up \( u \): Let \( u = x^2 \).
  2. Find \( du \):
    \( du = 2x \, dx \). We have \( x \, dx \) in our integral, so divide by 2: \( \frac{du}{2} = x \, dx \).
  3. Substitute and Integrate:
    \( \int \sinh(u) \left(\frac{du}{2}\right) = \frac{1}{2} \int \sinh(u) \, du \)
  4. Apply the Rule:
    \( = \frac{1}{2} \cosh(u) + C \)
  5. Final Answer:
    \( \frac{1}{2} \cosh(x^2) + C \)

Example 3: Rewriting for a Logarithm

Evaluate: \( \int \tanh(x) \, dx \)

There is no standard rule for standalone tanh! Rewrite it using its definition.

  1. Rewrite the Integrand:
    \( \int \frac{\sinh x}{\cosh x} \, dx \)
  2. Set up \( u \): Let the denominator be \( u \).
    \( u = \cosh x \)
  3. Find \( du \):
    \( du = \sinh x \, dx \). Notice that this perfectly matches the numerator!
  4. Substitute and Integrate: Use the Log Rule.
    \( \int \frac{1}{u} \, du = \ln|u| + C \)
  5. Final Answer:
    \( \ln(\cosh x) + C \)
    (Note: Absolute value bars can be dropped because cosh(x) is always positive.)

Example 4: Squared Functions

Evaluate: \( \int \text{sech}^2(3x - 1) \, dx \)

  1. Set up \( u \): Let \( u = 3x - 1 \).
  2. Find \( du \):
    \( du = 3 \, dx \implies \frac{du}{3} = dx \).
  3. Substitute and Integrate:
    \( \int \text{sech}^2(u) \left(\frac{du}{3}\right) = \frac{1}{3} \int \text{sech}^2(u) \, du \)
  4. Apply the Rule:
    \( = \frac{1}{3} \tanh(u) + C \)
  5. Final Answer:
    \( \frac{1}{3} \tanh(3x - 1) + C \)

Example 5: Using the Power Rule

Evaluate: \( \int \sinh^3(x) \cosh(x) \, dx \)

  1. Identify Pattern: We have a function raised to a power, multiplied by its derivative.
  2. Set up \( u \): Let \( u = \sinh x \).
  3. Find \( du \):
    \( du = \cosh x \, dx \).
  4. Substitute and Integrate: Apply the standard polynomial Power Rule.
    \( \int u^3 \, du = \frac{u^4}{4} + C \)
  5. Final Answer:
    \( \frac{1}{4} \sinh^4(x) + C \)

Example 6: Rational Argument

Evaluate: \( \int \frac{\text{csch}(1/x) \coth(1/x)}{x^2} \, dx \)

  1. Rewrite for Clarity:
    \( \int \text{csch}(x^{-1}) \coth(x^{-1}) \cdot x^{-2} \, dx \)
  2. Set up \( u \): Let \( u = x^{-1} \).
  3. Find \( du \):
    \( du = -x^{-2} \, dx \implies -du = x^{-2} \, dx \).
  4. Substitute and Integrate:
    \( \int \text{csch}(u) \coth(u) (-du) = -\int \text{csch}(u) \coth(u) \, du \)
  5. Apply the Rule: The integral of \( \text{csch } u \coth u \) is \( -\text{csch } u \).
    \( = -(-\text{csch } u) + C = \text{csch } u + C \)
  6. Final Answer:
    \( \text{csch}\left(\frac{1}{x}\right) + C \)
