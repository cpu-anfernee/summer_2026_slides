Chapter 8.4

Trigonometric Substitution

Transforming Algebraic Integrals into Trigonometric Integrals

The Three Substitutions $(a > 0)$

The objective with trigonometric substitution is to eliminate the radical in the integrand by using Pythagorean identities.

1. Pattern: $\sqrt{a^2 - u^2}$

  • Substitute: $u = a \sin \theta$ for $-\pi/2 \le \theta \le \pi/2$
  • Identity: $1 - \sin^2\theta = \cos^2\theta \implies \sqrt{a^2 - u^2} = a \cos \theta$

2. Pattern: $\sqrt{a^2 + u^2}$

  • Substitute: $u = a \tan \theta$ for $-\pi/2 < \theta < \pi/2$
  • Identity: $1 + \tan^2\theta = \sec^2\theta \implies \sqrt{a^2 + u^2} = a \sec \theta$

3. Pattern: $\sqrt{u^2 - a^2}$

  • Substitute: $u = a \sec \theta$
  • Identity: $\sec^2\theta - 1 = \tan^2\theta \implies \sqrt{u^2 - a^2} = \pm a \tan \theta$

Example 1: Sine Substitution

Evaluate: $\int \frac{dx}{x^2 \sqrt{9 - x^2}}$

  1. Identify and Substitute: Pattern is $\sqrt{a^2 - x^2}$ with $a = 3$.
    Let $x = 3 \sin \theta$. Then $dx = 3 \cos \theta \, d\theta$.
  2. Simplify the Radical: $\sqrt{9 - 9\sin^2\theta} = \sqrt{9\cos^2\theta} = 3 \cos \theta$.
  3. Rewrite the Integral:
    $\int \frac{3 \cos \theta \, d\theta}{(3 \sin \theta)^2 (3 \cos \theta)} = \int \frac{3 \cos \theta}{(9 \sin^2 \theta)(3 \cos \theta)} \, d\theta$
  4. Integrate: Cancel common factors.
    $= \frac{1}{9} \int \frac{1}{\sin^2 \theta} \, d\theta = \frac{1}{9} \int \csc^2 \theta \, d\theta = -\frac{1}{9} \cot \theta + C$
  5. Reference Triangle: Since $\sin \theta = \frac{x}{3} = \frac{\text{Opp}}{\text{Hyp}}$.
    The adjacent side is $\sqrt{9 - x^2}$.
    Therefore, $\cot \theta = \frac{\text{Adj}}{\text{Opp}} = \frac{\sqrt{9 - x^2}}{x}$.
  6. Final Answer:
    $-\frac{\sqrt{9 - x^2}}{9x} + C$

Example 2: Converting Limits of Integration

Evaluate: $\int_{\sqrt{3}}^{2} \frac{\sqrt{x^2 - 3}}{x} \, dx$

  1. Substitute: Pattern is $\sqrt{u^2 - a^2}$ with $u=x$, $a = \sqrt{3}$.
    Let $x = \sqrt{3} \sec \theta$. Then $dx = \sqrt{3} \sec \theta \tan \theta \, d\theta$.
  2. Simplify Radical: $\sqrt{3\sec^2\theta - 3} = \sqrt{3\tan^2\theta} = \sqrt{3} \tan \theta$.
  3. Convert Limits: Use $x = \sqrt{3} \sec \theta \implies \sec \theta = \frac{x}{\sqrt{3}}$.
    Lower limit: $x = \sqrt{3} \implies \sec \theta = 1 \implies \theta = 0$.
    Upper limit: $x = 2 \implies \sec \theta = \frac{2}{\sqrt{3}} \implies \theta = \frac{\pi}{6}$.
  4. Rewrite and Integrate:
    $\int_{0}^{\pi/6} \frac{\sqrt{3} \tan \theta}{\sqrt{3} \sec \theta} (\sqrt{3} \sec \theta \tan \theta) \, d\theta = \sqrt{3} \int_{0}^{\pi/6} \tan^2 \theta \, d\theta$
  5. Use Identity $\tan^2\theta = \sec^2\theta - 1$:
    $= \sqrt{3} \int_{0}^{\pi/6} (\sec^2 \theta - 1) \, d\theta = \sqrt{3} [\tan \theta - \theta]_0^{\pi/6}$
  6. Final Answer: Evaluate at the new limits.
    $= \sqrt{3} \left( \frac{1}{\sqrt{3}} - \frac{\pi}{6} \right) - (0) = 1 - \frac{\sqrt{3}\pi}{6}$

Example 3: Application to Arc Length

Find the arc length of the graph of $f(x) = \frac{1}{2}x^2$ from $x=0$ to $x=1$.

  1. Set up the Integral: $s = \int_{0}^{1} \sqrt{1 + [f'(x)]^2} \, dx$.
    Since $f'(x) = x$, we have $s = \int_{0}^{1} \sqrt{1 + x^2} \, dx$.
  2. Substitute: Let $x = \tan \theta \implies dx = \sec^2 \theta \, d\theta$.
    Radical becomes: $\sqrt{1 + \tan^2\theta} = \sec \theta$.
  3. Convert Limits:
    Lower: $x = 0 \implies \tan \theta = 0 \implies \theta = 0$.
    Upper: $x = 1 \implies \tan \theta = 1 \implies \theta = \pi/4$.
  4. Rewrite and Integrate:
    $\int_{0}^{\pi/4} (\sec \theta)(\sec^2 \theta) \, d\theta = \int_{0}^{\pi/4} \sec^3 \theta \, d\theta$
  5. Evaluate: Using the established formula for $\int \sec^3 \theta \, d\theta$:
    $= \frac{1}{2} [\sec \theta \tan \theta + \ln|\sec \theta + \tan \theta|]_0^{\pi/4}$
  6. Final Answer:
    $= \frac{1}{2} [\sqrt{2}(1) + \ln(\sqrt{2} + 1)] - \frac{1}{2}[0 + \ln(1)] = \frac{1}{2} [\sqrt{2} + \ln(\sqrt{2} + 1)]$

Example 4: Completing the Square

Evaluate: $\int \sqrt{x^2 - 2x} \, dx$

If the expression under the radical is a quadratic with a linear term, complete the square first!

  1. Complete the Square:
    $x^2 - 2x = (x^2 - 2x + 1) - 1 = (x - 1)^2 - 1$.
    The integral becomes $\int \sqrt{(x - 1)^2 - 1} \, dx$.
  2. Substitute: Pattern is $\sqrt{u^2 - a^2}$ with $u = (x - 1)$ and $a = 1$.
    Let $x - 1 = \sec \theta \implies dx = \sec \theta \tan \theta \, d\theta$.
  3. Simplify Radical: $\sqrt{\sec^2\theta - 1} = \tan \theta$.
  4. Rewrite and Integrate:
    $\int (\tan \theta)(\sec \theta \tan \theta) \, d\theta = \int \tan^2 \theta \sec \theta \, d\theta = \int (\sec^2 \theta - 1)\sec \theta \, d\theta$
    $= \int \sec^3 \theta \, d\theta - \int \sec \theta \, d\theta$
  5. Apply Formulas:
    $= \frac{1}{2}(\sec \theta \tan \theta + \ln|\sec \theta + \tan \theta|) - \ln|\sec \theta + \tan \theta| + C$
    $= \frac{1}{2}\sec \theta \tan \theta - \frac{1}{2}\ln|\sec \theta + \tan \theta| + C$
  6. Final Answer (Using Triangle):
    $= \frac{1}{2}(x - 1)\sqrt{x^2 - 2x} - \frac{1}{2}\ln|x - 1 + \sqrt{x^2 - 2x}| + C$
