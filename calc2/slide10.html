Chapter 7.1

Area Between Two Curves

Applications of Integration

Area with Vertical Rectangles (w.r.t. $x$)

If $f$ and $g$ are continuous on $[a, b]$ and $f(x) \ge g(x)$ for all $x$ in $[a, b]$, then the area of the region bounded by the graphs of $f$ and $g$ and the vertical lines $x = a$ and $x = b$ is:

\[ A = \int_a^b [f(x) - g(x)] dx \]

The "Top Minus Bottom" Rule

You can think of this as integrating the Top Curve minus the Bottom Curve. The beauty of this formula is that it works regardless of whether the curves are above or below the x-axis!

Area with Horizontal Rectangles (w.r.t. $y$)

Sometimes, finding the area is much easier if we integrate with respect to the y-axis. If curves are given as $x = f(y)$ and $x = g(y)$, and $f(y) \ge g(y)$ on the interval $[c, d]$:

\[ A = \int_c^d [f(y) - g(y)] dy \]

The "Right Minus Left" Rule

When integrating with respect to $y$, you subtract the curve on the left from the curve on the right.

Example 1: Given Limits

Find the area of the region bounded by $y = x^2 + 2$, $y = -x$, $x = 0$, and $x = 1$.

  1. Identify Top and Bottom: On $[0, 1]$, $x^2 + 2$ is clearly above $-x$.
    Top: $f(x) = x^2 + 2$. Bottom: $g(x) = -x$.
  2. Set up the Integral:
    $A = \int_0^1 [(x^2 + 2) - (-x)] dx = \int_0^1 (x^2 + x + 2) dx$
  3. Integrate:
    $\left[ \frac{x^3}{3} + \frac{x^2}{2} + 2x \right]_0^1$
  4. Evaluate:
    $\left( \frac{1}{3} + \frac{1}{2} + 2 \right) - 0 = \frac{2}{6} + \frac{3}{6} + \frac{12}{6}$
  5. Final Answer:
    $A = \frac{17}{6}$

Example 2: Finding Limits of Integration

Find the area bounded by the parabola $y = 2 - x^2$ and the line $y = -x$.

  1. Find Intersection Points: Set the equations equal.
    $2 - x^2 = -x \implies x^2 - x - 2 = 0 \implies (x-2)(x+1) = 0$
    Limits of integration are $x = -1$ to $x = 2$.
  2. Identify Top and Bottom: At $x=0$ (between limits), $y=2$ (parabola) and $y=0$ (line). Top is $2 - x^2$.
  3. Set up the Integral:
    $A = \int_{-1}^2 [(2 - x^2) - (-x)] dx = \int_{-1}^2 (-x^2 + x + 2) dx$
  4. Integrate: $\left[ -\frac{x^3}{3} + \frac{x^2}{2} + 2x \right]_{-1}^2$
  5. Evaluate at Limits:
    Upper (2): $-\frac{8}{3} + 2 + 4 = \frac{10}{3}$
    Lower (-1): $\frac{1}{3} + \frac{1}{2} - 2 = -\frac{7}{6}$
  6. Final Answer:
    $A = \frac{10}{3} - \left(-\frac{7}{6}\right) = \frac{20}{6} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2}$

Example 3: Curves that Cross

Find the area bounded by $f(x) = x^3$ and $g(x) = x$.

Warning: When curves cross, the "top" and "bottom" switch. We must split the integral!

  1. Find Intersections: $x^3 = x \implies x^3 - x = 0 \implies x(x-1)(x+1) = 0$.
    Intersections at $x = -1, 0, 1$. We have two regions.
  2. Analyze Regions:
    On $[-1, 0]$: Try $x = -0.5$. $f(-0.5) = -0.125$, $g(-0.5) = -0.5$. $x^3$ is Top.
    On $[0, 1]$: Try $x = 0.5$. $f(0.5) = 0.125$, $g(0.5) = 0.5$. $x$ is Top.
  3. Set up the Split Integrals:
    $A = \int_{-1}^0 (x^3 - x) dx + \int_0^1 (x - x^3) dx$
  4. Evaluate Region 1: $\left[ \frac{x^4}{4} - \frac{x^2}{2} \right]_{-1}^0 = 0 - \left(\frac{1}{4} - \frac{1}{2}\right) = \frac{1}{4}$
  5. Evaluate Region 2: $\left[ \frac{x^2}{2} - \frac{x^4}{4} \right]_0^1 = \left(\frac{1}{2} - \frac{1}{4}\right) - 0 = \frac{1}{4}$
  6. Final Answer: $A = \frac{1}{4} + \frac{1}{4} = \frac{1}{2}$

Example 4: Horizontal Representative

Find the area bounded by $x = 3 - y^2$ and $x = y + 1$.

Solving for $y$ here creates messy $\pm$ square roots. Integrate with respect to $y$ instead!

  1. Find Intersections (in terms of y):
    $3 - y^2 = y + 1 \implies y^2 + y - 2 = 0 \implies (y+2)(y-1) = 0$.
    Limits are $y = -2$ to $y = 1$.
  2. Identify Right and Left: Try $y=0$ (between limits).
    Parabola: $x = 3$. Line: $x = 1$. Parabola is the Right curve.
  3. Set up the Integral (Right - Left):
    $A = \int_{-2}^1 [(3 - y^2) - (y + 1)] dy = \int_{-2}^1 (2 - y - y^2) dy$
  4. Integrate: $\left[ 2y - \frac{y^2}{2} - \frac{y^3}{3} \right]_{-2}^1$
  5. Evaluate:
    Upper (1): $2 - \frac{1}{2} - \frac{1}{3} = \frac{7}{6}$
    Lower (-2): $-4 - 2 + \frac{8}{3} = -6 + \frac{8}{3} = -\frac{10}{3}$
  6. Final Answer: $A = \frac{7}{6} - \left(-\frac{20}{6}\right) = \frac{27}{6} = \frac{9}{2}$

Example 5: Trigonometric Curves

Find the area bounded by $y = \cos x$ and $y = \sin x$ from $x = 0$ to $x = \pi/4$.

  1. Identify Top and Bottom: On the interval $[0, \pi/4]$, the cosine wave starts at 1 and drops to $\sqrt{2}/2$. The sine wave starts at 0 and rises to $\sqrt{2}/2$.
    Therefore, $\cos x \ge \sin x$.
  2. Set up the Integral:
    $A = \int_0^{\pi/4} (\cos x - \sin x) dx$
  3. Integrate:
    $\left[ \sin x - (-\cos x) \right]_0^{\pi/4} = \left[ \sin x + \cos x \right]_0^{\pi/4}$
  4. Evaluate at Limits:
    Upper ($\pi/4$): $\sin(\pi/4) + \cos(\pi/4) = \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = \sqrt{2}$
    Lower (0): $\sin(0) + \cos(0) = 0 + 1 = 1$
  5. Final Answer:
    $A = \sqrt{2} - 1$

Example 6: Radicals & Lines

Find the area bounded by $y = \sqrt{x}$ and $y = x/2$.

  1. Find Intersections:
    $\sqrt{x} = \frac{x}{2} \implies x = \frac{x^2}{4} \implies x^2 - 4x = 0 \implies x(x-4) = 0$
    Limits are $x = 0$ to $x = 4$.
  2. Identify Top and Bottom: Test $x = 1$.
    $y = \sqrt{1} = 1$. $y = 1/2 = 0.5$. Top curve is $\sqrt{x}$.
  3. Set up the Integral:
    $A = \int_0^4 \left( x^{1/2} - \frac{x}{2} \right) dx$
  4. Integrate:
    $\left[ \frac{2}{3}x^{3/2} - \frac{x^2}{4} \right]_0^4$
  5. Evaluate at Limits:
    Upper (4): $\frac{2}{3}(4^{3/2}) - \frac{16}{4} = \frac{2}{3}(8) - 4 = \frac{16}{3} - \frac{12}{3} = \frac{4}{3}$
    Lower (0): $0 - 0 = 0$
  6. Final Answer:
    $A = \frac{4}{3}$
