Multiple Integrals

Basic Evaluation of Iterated Integrals

Evaluating Iterated Integrals

The fundamental rule for evaluating multiple integrals is to work from the inside out. Treat the variables of the outer integrals as constants while integrating the inner differential.

Double Integrals

Evaluate the inner integral with respect to $y$ (treating $x$ as a constant), then evaluate the result with respect to $x$.

\[ \int_a^b \left[ \int_{g_1(x)}^{g_2(x)} f(x,y) dy \right] dx \]

Triple Integrals

Evaluate with respect to $z$ first, then $y$, and finally $x$. The outermost limits must always be constants.

\[ \int_a^b \int_{g_1(x)}^{g_2(x)} \left[ \int_{h_1(x,y)}^{h_2(x,y)} f(x,y,z) dz \right] dy dx \]

Example 1: Double Integral (Constants)

Evaluate: $\int_0^2 \int_1^3 (x^2 y) dy dx$

  1. Evaluate the Inner Integral (w.r.t $y$): Treat $x^2$ as a constant multiplier.
    $\int_1^3 (x^2 y) dy = \left[ x^2 \frac{y^2}{2} \right]_1^3$
  2. Apply Inner Limits:
    $= x^2 \left( \frac{3^2}{2} - \frac{1^2}{2} \right) = x^2 \left( \frac{9}{2} - \frac{1}{2} \right) = x^2 \left( \frac{8}{2} \right) = 4x^2$
  3. Evaluate the Outer Integral (w.r.t $x$):
    $\int_0^2 4x^2 dx = \left[ \frac{4x^3}{3} \right]_0^2$
  4. Apply Outer Limits:
    $= \frac{4(2)^3}{3} - 0 = \frac{32}{3}$
  5. Final Answer:
    $\frac{32}{3}$

Example 2: Double Integral (Variables)

Evaluate: $\int_0^2 \int_0^x (4 - x - y) dy dx$

  1. Inner Integral (w.r.t $y$): Treat $4$ and $x$ as constants.
    $\int_0^x (4 - x - y) dy = \left[ 4y - xy - \frac{y^2}{2} \right]_0^x$
  2. Apply Inner Limits: Substitute $x$ in for $y$.
    $= \left( 4(x) - x(x) - \frac{x^2}{2} \right) - 0 = 4x - x^2 - \frac{x^2}{2}$
  3. Combine Terms:
    $= 4x - \frac{3x^2}{2}$
  4. Outer Integral (w.r.t $x$):
    $\int_0^2 \left( 4x - \frac{3x^2}{2} \right) dx = \left[ 2x^2 - \frac{x^3}{2} \right]_0^2$
  5. Apply Outer Limits:
    $= \left( 2(4) - \frac{8}{2} \right) - 0 = 8 - 4 = 4$
  6. Final Answer:
    $4$

Example 3: Trigonometric Limits

Evaluate: $\int_0^{\pi} \int_0^{\sin x} y \, dy dx$

  1. Inner Integral (w.r.t $y$):
    $\int_0^{\sin x} y \, dy = \left[ \frac{y^2}{2} \right]_0^{\sin x}$
  2. Apply Inner Limits:
    $= \frac{(\sin x)^2}{2} - 0 = \frac{1}{2} \sin^2 x$
  3. Outer Integral (w.r.t $x$): Use the half-angle identity $\sin^2 x = \frac{1 - \cos(2x)}{2}$.
    $\int_0^\pi \frac{1}{2} \sin^2 x \, dx = \frac{1}{2} \int_0^\pi \frac{1 - \cos(2x)}{2} dx = \frac{1}{4} \int_0^\pi (1 - \cos(2x)) dx$
  4. Integrate and Evaluate:
    $= \frac{1}{4} \left[ x - \frac{1}{2}\sin(2x) \right]_0^\pi$
  5. Apply Outer Limits:
    $= \frac{1}{4} \left( (\pi - 0) - (0 - 0) \right)$
  6. Final Answer:
    $\frac{\pi}{4}$

Example 4: Triple Integral (Constants)

Evaluate: $\int_0^1 \int_0^2 \int_0^3 (x y z) dz dy dx$

  1. Inner Integral (w.r.t $z$): Treat $xy$ as a constant.
    $\int_0^3 (x y z) dz = \left[ xy \frac{z^2}{2} \right]_0^3 = xy \left(\frac{9}{2}\right) = \frac{9}{2}xy$
  2. Middle Integral (w.r.t $y$): Treat $x$ as a constant.
    $\int_0^2 \frac{9}{2}xy dy = \left[ \frac{9}{2}x \frac{y^2}{2} \right]_0^2 = \frac{9}{4}x(4) = 9x$
  3. Outer Integral (w.r.t $x$):
    $\int_0^1 9x dx = \left[ \frac{9x^2}{2} \right]_0^1$
  4. Evaluate:
    $\frac{9(1)^2}{2} - 0 = \frac{9}{2}$
  5. Final Answer:
    $\frac{9}{2}$

Example 5: Triple Integral (Variables)

Evaluate: $\int_0^1 \int_0^x \int_0^{x+y} 2z \, dz dy dx$

  1. Inner Integral (w.r.t $z$):
    $\int_0^{x+y} 2z \, dz = \left[ z^2 \right]_0^{x+y} = (x+y)^2 - 0$
    Expand: $x^2 + 2xy + y^2$
  2. Middle Integral (w.r.t $y$): Integrate the expanded polynomial.
    $\int_0^x (x^2 + 2xy + y^2) dy = \left[ x^2y + x(y^2) + \frac{y^3}{3} \right]_0^x$
  3. Apply Middle Limits: Substitute $x$ for $y$.
    $= (x^2(x) + x(x^2) + \frac{x^3}{3}) - 0 = x^3 + x^3 + \frac{x^3}{3} = \frac{7}{3}x^3$
  4. Outer Integral (w.r.t $x$):
    $\int_0^1 \frac{7}{3}x^3 dx = \left[ \frac{7}{12}x^4 \right]_0^1$
  5. Final Answer:
    $\frac{7}{12}$

Example 6: Evaluating from Inside Out

Evaluate: $\int_0^1 \int_0^z \int_0^y 12x \, dx dy dz$

  1. Inner Integral (w.r.t $x$):
    $\int_0^y 12x \, dx = \left[ 6x^2 \right]_0^y = 6y^2$
  2. Middle Integral (w.r.t $y$):
    $\int_0^z 6y^2 \, dy = \left[ 2y^3 \right]_0^z$
  3. Apply Middle Limits: Substitute $z$ for $y$.
    $= 2z^3 - 0 = 2z^3$
  4. Outer Integral (w.r.t $z$):
    $\int_0^1 2z^3 \, dz = \left[ \frac{1}{2}z^4 \right]_0^1$
  5. Evaluate:
    $= \frac{1}{2}(1)^4 - 0$
  6. Final Answer:
    $\frac{1}{2}$
