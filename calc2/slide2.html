Chapter 4.1

Antiderivatives and Indefinite Integration

Part 2: Trigonometric Functions

From Derivatives to Antiderivatives

Just as with algebraic functions, the integration rules for trigonometric functions are derived directly by reversing their derivative rules.

Derivatives

\(\frac{d}{dx}[\sin x] = \cos x\)

\(\frac{d}{dx}[\cos x] = -\sin x\)

\(\frac{d}{dx}[\tan x] = \sec^2 x\)

Antiderivatives

\(\int \cos x dx = \sin x + C\)

\(\int \sin x dx = -\cos x + C\)

\(\int \sec^2 x dx = \tan x + C\)

Watch the negative signs! Integrating sine results in a NEGATIVE cosine.

The 6 Basic Trigonometric Rules

\(\int \sin x~dx=-\cos x+C\)

\(\int \cos x~dx=\sin x+C\)

\(\int \sec^{2}x~dx=\tan x+C\)

\(\int \sec x\tan x~dx=\sec x+C\)

\(\int \csc^{2}x~dx=-\cot x+C\)

\(\int \csc x\cot x~dx=-\csc x+C\)

Step-by-Step Example 1

Evaluate: \(\int 2\sin x~dx\)

  1. Apply the Constant Multiple Rule: Factor the 2 outside the integral to isolate the trigonometric function.
    \(= 2\int \sin x~dx\)
  2. Apply the Trig Integration Rule: Recall that the antiderivative of \(\sin x\) is \(-\cos x\).
    \(= 2(-\cos x) + C\)
  3. Simplify: Multiply the constant back in.

    Final Answer: \(= -2\cos x + C\)

Step-by-Step Example 2

Evaluate: \(\int (t^{2} - \sin t)dt\)

  1. Apply the Sum/Difference Rule: Separate the polynomial and the trigonometric term into two integrals.
    \(= \int t^{2}dt - \int \sin t~dt\)
  2. Integrate Term by Term:
    For \(\int t^{2}dt\), use the Power Rule: \(\frac{t^{3}}{3}\).
    For \(\int \sin t~dt\), use the Trig Rule: \(-\cos t\).
    \(= \left(\frac{t^{3}}{3}\right) - (-\cos t) + C\)
  3. Simplify Signs: Notice the double negative \(- (-\cos t)\).

    Final Answer: \(= \frac{t^{3}}{3} + \cos t + C\)

Step-by-Step Example 3

Evaluate: \(\int \frac{\sin x}{\cos^{2}x}dx\)

There is no rule for this exact fraction. We must rewrite it using trigonometric identities first!

  1. Split the Denominator: Rewrite \(\cos^{2}x\) as \(\cos x \cdot \cos x\) and separate the fraction.
    \(= \int \left(\frac{1}{\cos x}\right)\left(\frac{\sin x}{\cos x}\right)dx\)
  2. Apply Identities: Convert to known functions.
    Recall: \(\frac{1}{\cos x} = \sec x\) and \(\frac{\sin x}{\cos x} = \tan x\).
    \(= \int \sec x \tan x~dx\)
  3. Integrate using Basic Rules: Now it matches one of our 6 standard rules.

    Final Answer: \(= \sec x + C\)

Step-by-Step Example 4 (IVP)

Find the particular solution to \(f^{\prime}(x) = \sec^{2}x\) given the initial condition \(f\left(\frac{\pi}{4}\right) = 3\).

  1. Find the General Antiderivative:
    \(f(x) = \int \sec^{2}x~dx = \tan x + C\)
  2. Substitute the Initial Condition: Plug in \(x = \frac{\pi}{4}\) and set it equal to 3.
    \(3 = \tan\left(\frac{\pi}{4}\right) + C\)
  3. Solve for C: We know \(\tan\left(\frac{\pi}{4}\right) = 1\).
    \(3 = 1 + C \implies C = 2\)
  4. Write the Particular Solution: Replace C in the general equation.

    Final Answer: \(f(x) = \tan x + 2\)
